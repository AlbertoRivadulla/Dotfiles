# Dotfiles
My dotfiles for Linux and MacOS

## Installation

- Install the following software
    - Homebrew (on MacOS)
    - Kitty
    - Neovim
    - Powerlevel10k theme for zsh [link](https://github.com/romkatv/powerlevel10k)
- Run the script `install.sh`
